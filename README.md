# mailchimp-test


## To Run  

### Install Protractor (globally)
npm install -g protractor  

### Update Selenium Server  
webdriver-manager update  

### Start Selenium Server
webdriver-manager start
  
  
# Run Tests

### Run Protractor version 
```
protractor conf.js
```

### Run python version
```
python LeadershipTeamToCsv.py
```
